# 🎙️ VoiceFlow — הקלטה ותמלול קולי

אפליקציית ווב שמאפשרת להקליט את עצמך והטקסט מתומלל אוטומטית בזמן אמת — בסגנון [Wispr Flow](https://wisprflow.ai).
עובדת ישירות בדפדפן, ללא התקנה וללא שרת.

## ✨ מה יש כאן

- 🎤 **הקלטה בלחיצה אחת** — כפתור מיקרופון גדול, עם אנימציית הקלטה וחיווי זמן.
- ⚡ **תמלול בזמן אמת** — הטקסט מופיע תוך כדי דיבור (Web Speech API המובנה בדפדפן).
- 🌍 **רב-לשוני** — עברית, אנגלית, ערבית, רוסית, ספרדית, צרפתית.
- 🧠 **מנוע דיוק גבוה (אופציונלי)** — חיבור ל-OpenAI Whisper באמצעות מפתח API אישי.
- 📊 **ויזואליזציה של גלי הקול** בזמן ההקלטה.
- 📋 **העתקה / ⬇️ הורדה / 🧹 ניקוי טקסט** (פיסוק ורווחים).
- 🕘 **היסטוריה** — כל תמלול נשמר מקומית (localStorage) וניתן לטעון אותו מחדש.
- ⌨️ **Push-to-talk** — החזקת מקש הרווח מקליטה כל עוד הוא לחוץ.
- 📌 **העתקה אוטומטית** — בסיום ההקלטה הטקסט מועתק אוטומטית, מוכן להדבקה בכל מקום.

## 💬 איך מדביקים את התמלול בקלוד / Gmail / כל מקום

אפליקציית ווב לא יכולה להזריק טקסט אוטומטית לתוך אתרים אחרים (חסימת אבטחה של הדפדפן),
אז הזרימה היא:

1. מקליטים ב-VoiceFlow ומדברים.
2. מפסיקים — הטקסט מועתק אוטומטית ל-clipboard (אם האפשרות "העתק אוטומטית" מסומנת).
3. עוברים לקלוד / Gmail / וכו' ומדביקים עם **Ctrl+V** (או **Cmd+V** ב-Mac).

> רוצים שהתמלול יוקלד **אוטומטית** לתוך כל שדה (כמו Wispr Flow)? לשם כך צריך תוסף Chrome
> או אפליקציית שולחן עבודה — אפשר לבנות גם את זה בנפרד.

## 🚀 הפעלה

פשוט פותחים את `index.html` בדפדפן **Chrome / Edge** (מומלץ, בגלל תמיכה מלאה בתמלול מובנה):

```bash
# מהתיקייה voice-transcriber
# אפשרות א': פתיחה ישירה
open index.html        # macOS
xdg-open index.html    # Linux
start index.html       # Windows

# אפשרות ב' (מומלץ — חלק מהדפדפנים דורשים origin בטוח למיקרופון):
python3 -m http.server 8000
# ואז גשו אל http://localhost:8000
```

בפעם הראשונה הדפדפן יבקש הרשאת גישה למיקרופון — יש לאשר.

## 🧠 מנוע Whisper (דיוק גבוה)

1. לחצו על **⚙️ הגדרות מפתח API** בסרגל הצד.
2. הזינו מפתח OpenAI API (`sk-...`). המפתח נשמר **מקומית בלבד** בדפדפן ונשלח ישירות ל-OpenAI.
3. בחרו במנוע **OpenAI Whisper** מהרשימה.

במצב זה ההקלטה מוקלטת במלואה ונשלחת ל-Whisper בסיום — מדויק יותר, אך לא בזמן אמת.

## 🔒 פרטיות

- במנוע המובנה — הכל נשאר בדפדפן (התמלול מתבצע ע"י מנוע הדפדפן/מערכת ההפעלה).
- במנוע Whisper — האודיו נשלח ישירות ל-OpenAI בלבד, עם המפתח שלכם.
- ההיסטוריה והמפתח נשמרים ב-`localStorage` המקומי, ללא שרת חיצוני.
